# security-monitoring-detection-lab

## Overview
Simulated an enterprise SOC environment to monitor and detect security threats.

## Tools Used
- ELK Stack / Splunk
- Zeek / Suricata
- Windows & Linux VMs

## Attack Scenarios
- Brute-force login attempts
- Malware activity (EICAR)
- Anomalous network traffic

## Detections
- Excessive failed logins
- Suspicious outbound traffic
- Malware hash detection

## Outcome
Created dashboards and alerts for real-time monitoring.
